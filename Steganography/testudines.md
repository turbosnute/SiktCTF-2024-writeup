## Solution

There is penup() code in the data of the image. Use the `strings` command to extract code from the image and use a tool like https://trinket.io/python to run the code and see the flag.

```bash
$ strings turtle.jpeg
...

e"UQR

&)qKK@

left(180)
goto(-200, -10)
left(180)
...
left(90)
forward(3)
penup()
goto(180, -10)
```
